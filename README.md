## Usage

1. [下载 flow](https://github.com//MuYunyun/commonSearch/raw/master/flow.alfredworkflow)
2. 双击下载好的 `flow.alfredworkflow` 文件, 自动完成安装。

> 使用该插件需要安装 node 环境
